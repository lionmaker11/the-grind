# Skill: Morning Brief

## Trigger
T.J. says "good morning" or this runs at 6:00 AM heartbeat.

## Process
1. Read [[MEMORY]] master index for current priorities and patterns to watch.
2. Check today's day of week → load today's tasks from Basecamp.
3. Check Google Calendar for today's meetings/appointments.
4. Check Linear for open tickets on active projects (MARCUS, BiggerSpreads if active).
5. Check Week A/Week B toggle → does [[aurelia|Aurelia]] need school drop-off today? (7:30 AM departure, 8:10 drop, 20 min drive)
6. Build morning brief with:
   - Gratitude prompt
   - Today's Pomodoros with concrete "done" conditions
   - Recurring tasks ([[708-pallister-status|708 Pallister]] check-in, workout if Mon/Wed/Fri, [[motor-city-deals-status|Motor City Deals]] if Mon/Thu)
   - Category balance (which of the 7 categories get attention today?)
   - Active project tracker (which projects are being touched this week?)
   - Any financial reminders (bills, deadlines Chief is tracking)
7. Push to Telegram.

## Output Format
Keep it to one phone screen max. Pomodoros numbered. Done conditions specific.

## Chain
If meetings today → invoke [[meeting-prep]] skill for each.
