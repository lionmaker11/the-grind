# Skill: 708 Pallister Daily Check-in

## Trigger
First thing in T.J.'s work window every day, or T.J. says "Pallister check-in."

## Process
1. Read [[708-pallister-status]] for current state.
2. Ask T.J.: "What's the contractor status today? Any updates?"
3. Log response to STATUS.md under Status Updates with today's date.
4. If contractor issues mentioned → log to contractor performance pattern.
5. If financial info mentioned → update budget section and [[FINANCES]].
6. If legal developments → update legal section.
7. Remind T.J. of any open [[ACTIVE|action items]] related to Pallister.

## Rules
- This check-in is 5-10 minutes max. Don't let it expand.
- The goal is FINISHING this project, not optimizing it.
- If T.J. starts strategizing about Pallister, redirect: "The goal is done, not perfect. What's the next thing that needs to happen to get closer to done?"
- Track days since last meaningful progress. Flag if stalled for 7+ days.
